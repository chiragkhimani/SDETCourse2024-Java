4. Static Keyword

A. Static Variable
1. Static keyword we need to use to  create static
2. Class level variable, so we can access it directly with ClassName
3. It's having only ONE copy at the class level (so every object will share that memory)
4. If we change the value once for a static variable, it will use that value for all the objects.

Real Examples:
1. Company Name for all employees
2. Brand of any car
3. Collage name for all the students

B. Static Method
1 Static Method belongs to the class and we can directly access it with the ClassName
2 Under static method -->> we can access ONLY static Variables
          	But,
If the method is NOT static -->> then we can access both static and non-static variable
